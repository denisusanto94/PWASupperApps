Semua gambar undangan menggunakan format .webp.

File yang dipakai di undangan:
- bg_utama.webp   : Background cover (Open the Invitation)
- bg_creme.webp   : Background hero (nama mempelai)
- mempelai_pria.webp  : Foto mempelai pria (lingkaran)
- mempelai_wanita.webp : Foto mempelai wanita (lingkaran)
- gebyok_atas.webp    : Dekorasi gebyok atas
- pohon.webp          : Dekorasi pohon (animasi sway)
- frame_border.webp    : Bingkai dekoratif Bride & Groom

Ukuran disarankan: background minimal 1200px lebar, foto couple minimal 300x300px.
Format: .webp (untuk performa dan konsistensi tampilan).
